## ğŸ¯ IMPORTANT CONCEPTS

### Logistic Regression
- **Binary classification**: Predicts 0 or 1
- **Sigmoid function**: Outputs probability (0-1)
- **Interpretable**: Can see feature importance
- **Fast**: Good for real-time predictions

### Why StandardScaler?
- Normalizes all features to same scale
- Prevents large-range features from dominating
- Required for Logistic Regression to work well
- Formula: (x - mean) / std_dev

### Train-Test Split
- **80-20**: 80% training, 20% testing
- **Why**: Evaluate on unseen data
- **Stratify**: Keep class distribution same
- **Prevents overfitting**: Fair assessment

### Model Persistence with Joblib
- Save: `joblib.dump(model, 'name.pkl')`
- Load: `model = joblib.load('name.pkl')`
- **Why**: Reuse without retraining
- **Medical context**: Always scale with same scaler!
